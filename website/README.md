# Website

One-page landing for Production Hybrid.

## Local preview

Open `index.html` in a browser, or:

```bash
npx serve website
```

## Deploy to GitHub Pages

GitHub Pages only supports `/` or `/docs`. Options:

1. **Root deploy:** copy `index.html` to repo root (or use `website/` via Actions)
2. **Custom domain:** point `productionhybrid.dev` to GitHub Pages when ready
3. **Now:** host on GitHub repo homepage + `website/index.html` for local preview
